SunTree

Custom designed Family Tree in the shape of the Sun

Open source renderer which produces web-based visualization
and print version for up to A0 size.

See example

Create your Family Tree in 3 steps

Prepare the data

There are 2 options to fill in the information about your family.

  • Create the tree on familyecho.com. It's fast and easy to use web service. Once finished, export the tree in CSV format.
  • The other option is to manually create CSV file. See example here.

Create the layout

To visualize the data, you need to copy the source code.

  • Fork our GitHub repository.
  • Replace /src/assets/tree-data-en.csv with exported file from FamilyEcho.
  • Open repository settings and publish it as a GitHub Page. You can even connect a custom domain name.

Customize for your needs

SunTree comes with multi-language support, print version, blocks for meta information and customizable layout. It also works well on mobile phones.

Check out the documentation and feel free to reach out us if you have any questions.
